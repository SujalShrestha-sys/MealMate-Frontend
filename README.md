# 📌 Project Introduction

MealMate is a **full-stack meal subscription and food ordering platform** developed to streamline and modernize the dining experience in campus environments. The system addresses common challenges such as long queues, inefficient manual ordering, lack of structured meal planning, and limited access to organized food services.

The platform enables students to **browse dishes, search and filter menus, subscribe to meal plans, and place orders seamlessly** through a responsive and intuitive interface. By combining **on-demand ordering with subscription-based meal planning**, MealMate ensures both flexibility and efficiency for users and service providers.

A key feature of MealMate is its **subscription-based model**, offering weekly and monthly plans tailored to different dietary needs and lifestyles. This approach simplifies daily decision-making for students while helping vendors **predict demand, reduce waste, and optimize operations**.

---

## 🧠 Technology Stack

- **Frontend:** React, Tailwind CSS  
- **Admin UI:** Material UI (MUI)  
- **Backend:** Node.js, Express.js  
- **Database:** PostgreSQL  
- **ORM:** Prisma  

---

## 🔐 Security & Authentication

- JWT-based authentication  
- Access & Refresh token mechanism  
- Role-based authorization  
- Password hashing using bcrypt  

---

## ⚙️ Core Features

- **Menu Management:** Browse, search, and filter dishes  
- **Subscription Plans:** Weekly and monthly meal plans  
- **Cart & Ordering:** Add items, manage quantities, and place orders  
- **Pickup Scheduling:** Select time slots to reduce congestion  
- **Payment Integration:** Khalti digital payment gateway  
- **Email Services:** Resend for password reset and notifications  
- **Feedback System:** Ratings and reviews for dishes  
- **Inventory Tracking:** Manage ingredients and stock levels  
- **Notifications:** Real-time system updates  
- **Chat System:** Built-in communication support
